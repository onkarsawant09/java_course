## 🚀 How to Run

### Prerequisites
- **Java Development Kit (JDK)** 8 or higher installed.
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or a simple text editor + terminal.

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/java-practice.git
   cd java-practice
   ```

2. **Compile and run any program**
   - Using terminal:
     ```bash
     javac basics/HelloWorld.java
     java basics.HelloWorld
     ```
   - Or open the folder in your favorite IDE and run the `.java` file directly.

## 📚 Topics Covered

| Topic | Description |
|-------|-------------|
| **Basics** | Variables, data types, operators, loops (for, while, do-while), conditionals (if-else, switch) |
| **Object-Oriented Programming** | Classes, objects, constructors, `this` keyword, inheritance, polymorphism (overloading, overriding), encapsulation, abstraction (abstract classes, interfaces) |
| **Arrays** | Single and multi‑dimensional arrays, array manipulation, `Arrays` utility class |
| **Strings** | String immutability, `StringBuilder`, `StringBuffer`, common methods (`charAt`, `substring`, `split`, etc.) |
| **Collections Framework** | `List` (ArrayList, LinkedList), `Set` (HashSet, TreeSet), `Map` (HashMap, TreeMap), `Queue` (PriorityQueue) |
| **Exception Handling** | `try-catch-finally`, `throw`, `throws`, custom exceptions, try‑with‑resources |
| **File I/O** | Reading/writing text files (`FileReader`, `BufferedReader`, `FileWriter`), binary files (`ObjectInputStream`, `ObjectOutputStream`) |
| **Multithreading** | Extending `Thread` class, implementing `Runnable`, `synchronized` keyword, inter‑thread communication |
| **Data Structures (Custom)** | Stack (using array & linked list), Queue, Singly/Doubly Linked List, Binary Search Tree (basic) |
| **Algorithms** | Sorting (Bubble, Selection, Insertion, Merge, Quick), Searching (Linear, Binary), Recursion (factorial, Fibonacci, Tower of Hanoi) |
| **Patterns** | Star patterns, number patterns (pyramid, diamond, Floyd’s triangle) |
| **Mini Projects** | Simple projects like ATM simulation, Student Management System, Calculator, Tic-Tac-Toe, etc. |
